## 🚀 Sobre o projeto

A **Padaria Aurora** foi criada como um projeto de estudo e portfólio, desenvolvido utilizando **React**, **TypeScript** e **Bootstrap** com o objetivo de praticar desenvolvimento front-end moderno, responsividade e construção de interfaces atrativas para negócios locais.<br>

O projeto apresenta uma interface inspirada em padarias e cafeterias modernas, destacando produtos, ambiente e identidade visual de forma profissional.<br>

O foco principal do projeto foi praticar:

* Responsividade
* Estruturação de layouts
* Componentização visual
* Estilização moderna
* Utilização do Bootstrap
* Organização de código front-end

## ⚙️ Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/rafaelottdev/padaria-aurora.git
```

### Acesse a pasta do projeto

```bash
cd padaria-aurora
```

### Instale as dependencias

```bash
npm install
```

### Execute o projeto

```bash
npm run dev
```
